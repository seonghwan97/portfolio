CS401 Lab1: Circle Area Calculator

- How to Run the Program
	1. In Eclipse:
		1) Open the project in Eclipse.
		2) Run the MainApplication.
		3) When the prompt "Enter the radius of the circle:" appears, input the radius.
		4) If a non-numeric value is entered, you will be asked to enter a valid numeric value.
		5) Once a valid radius is entered, the area of the circle will be displayed.

	2. JAR File Execution:
		1) From the command line, navigate to the directory containing the JAR file.
		2) Run the following command:

			java -jar MainApplication.jar

		3) The same steps 3-5 from the "In Eclipse" section will appear
